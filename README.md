# Useful Scripts

This repository consists some useful scripts while setting up a server.

## How to Use

- sudo curl -sL [link of file] -o setup.sh
- chmod 755 setup.sh
- ./setup.sh [extra arguments]

## Options

- Install Node
- Install Nginx
- Configure HTTPS [email] [domain-name]
